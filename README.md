# loading-anim

Небольшая анимация загрузки главного экрана (3D-лоадер). Vue 2 + Vite.

## Запуск

```bash
npm install
npm run dev
```

Сборка: `npm run build`. Просмотр сборки: `npm run preview`.

## Структура

```
loading-anim/
├── index.html
├── vite.config.js
├── package.json
├── src/
│   ├── main.js
│   ├── App.vue
│   └── components/Loader.vue
└── README.md
```
